Template Variables

You can use the following variables in your message templates:

  • {name} - The recipient's first name (or username if no first name)
  • {group_link} - The target group link or identifier
  • {username} - The recipient's username (if available)
  • {first_name} - The recipient's first name
  • {last_name} - The recipient's last name
Tip: Keep messages personal and avoid spam-like content to improve delivery rates.
