Campaign Information
How it works:
  1. Upload a CSV file with member data
  2. Choose an account to send from
  3. Select action: invite to group or send DM
  4. Choose a message template
  5. Configure limits and timing
  6. Start the campaign
Rate Limits: Telegram has strict rate limits. Use appropriate intervals to avoid restrictions.
CSV Format

Your CSV file should include these columns:

  • ID (required)
  • Username
  • First Name
  • Last Name
  • Phone
  • Access Hash
You can export CSV files from completed scraping sessions.
